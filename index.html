Полный комплект дизайн проекта включает в себя:

  • обмеры с площадями;
  • план с расстановкой мебели;
  • план перепланировки;
  • план потолков с отметками;
  • план полов с раскладками материалов;
  • план осветительной сети с привязками светильников и выключателей, маркировка групп освещения;
  • спецификация осветительных приборов с рисунками приборов;
  • план розеточной сети с привязкой розеток и указанием блоков.

картинки интерьеров по каждой комнате с описанием применяемых материалов;
чертежи интерьеров по каждой комнате (развертки по стенам с габаритами мебели, указание отделочных материалов на развертках);
спецификация отделочных материалов (обои, плитка);
подбор мебели, выбор материалов (предложение по эскизам с сайтов интернет -магазинов, выполняются первоначальные эскизы, при встрече утверждаем выбранные материалы в магазине или другие, выполняю окончательные картинки, выпускаем проект).

Проект выдается на руки в 2-х экземплярах формата А4 с DVD диском на котором все материалы в ходе выполнения проекта и чертежи выпуска проекта

Cтоимость полного комплекта 10$/м2, аванс 30-50%, окончательный расчет при выдаче распечатанного проекта

Работа при выдаче проекта не прекращается до полного завершения ремонта до установки мебели
